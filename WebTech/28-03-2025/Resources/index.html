Methods:

  1. split();

    2. The split() converts the string to array. It has two parameters.

    1. Empty Parameter

      2. It converts the entire string to a single element of array stored at index 0.

              const str = "hello"
        console.log(str.split())

    2. Null String as parameter

      3. It converts the letters of a string to element of array with begin index 0 and ending index with string leangth-1. 

              const str = "hello"
        console.log(str.split(""))

    3. Special Symbols within the String

      4. It splits the string to number of elements of array where it found @.

              const str2 = "hello@sudhansu@panda"
        console.log(str2.split("@"))
  2. push()

    3. The push() is used to add new elements into the ending index of an array. 

              const str = [1,2,3,4,5,6,7]
          console.log(str)
          str.push(8,9,10,11)
          console.log(str)
  3. unshift();

    4. The unshift() is used to add new elements into the starting index of an array. 

              const str = [1,2,3,4,5,6,7]
          console.log(str)
          str.unshift(-2,-1,1,0)
          console.log(str)
  4. pop();

    5. The pop() is used to remove a single element from the ending index of an array.

              const str = [1,2,3,4,5,6,7]
          console.log(str)
          str.pop()
          console.log(str)
  5. shift();

    6. The shift() is used to remove a single element from the starting index of an array.

              const str = [1,2,3,4,5,6,7]
          console.log(str)
          console.log(str.shift())
          console.log(str)
  6. slice()
    7.           const str = [1,2,3,4,5,6,7]
          console.log(str)
          console.log(str.slice(2,5))
          console.log(str)
  7. join()
