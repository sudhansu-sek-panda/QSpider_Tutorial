String

It is a predefined class present in Java.lang package and considered to be most important class after object class.

Importance of String class

  • It is the mandatory class used in main method argument.

Properties of String class

  • String class objects are immutable

    
            public class Demo {
                public static void main(String[] args) {
                  String s = new String("Ganji");
                  System.out.println(s);
                  s.concat("Chudel");
                  System.out.println(s); 
                }  
              
              }
    Output: 
                Ganji
            Ganji
    If we're creating object of string class by passing a content then we can't perform any kind of changes or updation with the content of the object.

    • Twice Ganji came because of immutable nature of the String.

    Note:

    The Object 'GanjiChudel' is still present interanlly.

  • In string class toString() method has already been overriden to return the content of the object nor the address of the object
